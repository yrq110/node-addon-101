# node-addon-101

* C++: [node-addon-api](https://github.com/nodejs/node-addon-api)
* env: node-12.9
* build: [node-gyp](https://github.com/nodejs/node-gyp) & binding.gyp

## Commands

```shell
$ yarn install
$ yarn run build
$ node index.js
```
